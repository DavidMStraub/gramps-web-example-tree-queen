# Gramps Web Example Tree – Queen

An example family tree database for [Gramps](https://gramps-project.org/) and [Gramps Web](https://www.grampsweb.org/), centered on the British royal family.

> **Disclaimer:** This dataset is intended solely to demonstrate the features of Gramps and Gramps Web. It makes no claim to genealogical or factual accuracy and should not be used as a historical or genealogical reference.

## Contents

- `queen.gramps` – the Gramps XML database
- `media/` – associated media files (portraits, photographs, coats of arms)

## Data Sources & Licensing

This dataset is assembled from multiple sources with different licenses. The overall dataset is governed by **Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)**, driven by the ShareAlike requirement of Wikipedia-sourced content.

### Genealogical data

Core genealogical data (names, dates, relationships) is adapted from the GEDCOM samples at <https://github.com/D-Jeffrey/gedcom-samples> and is dedicated to the public domain under **CC0 1.0 Universal**.

### Imagery and encyclopedic content

Biographical text excerpts, portraits, and other media are sourced from [Wikipedia](https://www.wikipedia.org/) and Wikimedia Commons under **CC BY-SA 4.0**, which requires attribution to the original contributors and ShareAlike distribution of any derivatives. Gramps **citations** on individual records flag content carrying a specific CC license; check them before reusing a record.
